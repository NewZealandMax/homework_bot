## Telegram-бот

### Описание проекта:
Направляет информацию в Telegram о состоянии проверки домашних работ на курсе обучения

### Запуск проекта
```
python homework.py
```

### Версия языка
`Python 3.9.2`

Зависимости представлены в файле `requirements.txt`

### Стек технологий
`requests` `telegram` `dotenv`
